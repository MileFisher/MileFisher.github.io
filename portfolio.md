#### Guardrail LMS
**Tech:** React, Vite, Node.js, Express, PostgreSQL, Render, Cloudflare Pages, OpenRouter

Guardrail LMS is a process-based academic integrity platform built around two core ideas: an integrity monitor and an AI-guided tutor. The system combines keystroke-dynamics telemetry, signed events, and anomaly detection with a separate Socratic learning assistant for non-assessed study sessions. The project also demonstrates full-stack deployment with a React frontend, an Express API backend, and a managed PostgreSQL database.

GitHub: https://github.com/MileFisher/Guardrail_LMS

#### AI Clinical Navigator

I include this project because it reflects both my contribution to the user-facing experience and my involvement in a larger AI healthcare workflow that is still in progress.

GitHub: https://github.com/qchau0202/ai-clinical-navigator

#### Advanced DB Final
**Tech:** Python, FastAPI, SQL Server, MongoDB, Stripe, multi-service architecture

This project shows my exposure to database-heavy system design, cross-service coordination, and combining SQL and NoSQL technologies in one application.
